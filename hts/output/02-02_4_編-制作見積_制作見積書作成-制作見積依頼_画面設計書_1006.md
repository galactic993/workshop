## イベント記述書

### EVENT0001
- **トリガー**: 初期表示時
- **区分**: JavaScript
- **処理メイン**: 1.JavaScript有効化判定
2．権限判定
3.見積情報取得

### EVENT0002
- **トリガー**: 制作見積依頼 - 担当部署(▶ボタン)押下時
- **区分**: JavaScript
- **処理メイン**: 1.展開/折り畳み

### EVENT0003
- **トリガー**: 制作見積依頼 - 担当部署選択（入力）時
- **区分**: JavaScript
- **処理メイン**: 1.部署選択
2.入力チェック

### EVENT0004
- **トリガー**: 制作見積依頼 - 依頼概要入力時
- **区分**: JavaScript
- **処理メイン**: 1.入力チェック

### EVENT0005
- **トリガー**: 制作見積依頼 - 参考資料入力時
- **区分**: JavaScript
- **処理メイン**: 1.入力チェック

### EVENT0006
- **トリガー**: 制作見積依頼 - 「削除(アイコン)」ボタン押下時
- **区分**: JavaScript
- **処理メイン**: 1.項目削除

### EVENT0007
- **トリガー**: 制作見積依頼 - 「依頼フォーム追加」ボタン押下時
- **区分**: JavaScript
- **処理メイン**: 1.項目追加

### EVENT0008
- **トリガー**: 「依頼登録」ボタン押下時
- **区分**: JavaScript
- **処理メイン**: 1.入力チェック
2.情報登録/更新
3.メール送信

### EVENT0009
- **トリガー**: 「一覧へ戻る」ボタン押下時
- **区分**: JavaScript
- **処理メイン**: 1.画面遷移
