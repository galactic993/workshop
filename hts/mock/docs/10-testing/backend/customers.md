# 得意先APIテスト仕様

## 概要

得意先検索・管理APIのテスト。2つのテストファイルに分かれている:

- `CustomerSearchApiTest.php`: 見積用得意先検索API
- `SectionCustomerApiTest.php`: 部署別得意先管理API

---

## CustomerSearchApiTest.php

**ファイル**: `backend/tests/Feature/CustomerSearchApiTest.php`

見積機能で使用する得意先検索APIのテスト。

### エンドポイント

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/api/quotes/customers/suggest` | 得意先サジェスト |
| GET | `/api/quotes/customers/search` | 得意先検索 |

### GET /api/quotes/customers/suggest

#### 認証・権限チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは得意先サジェストを取得できない | セッションなしでアクセス拒否 | 401 |
| 権限のないユーザーは得意先サジェストを取得できない | 見積権限なしで拒否 | 403 |
| アクセス区分00（全て）のユーザーは取得できる | 管理者はアクセス可能 | 200 |
| sales.quotes.create権限を持つユーザーは取得できる | 見積権限ありでアクセス可能 | 200 |

#### バリデーション

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| queryパラメータは必須 | query未指定でエラー | 422 |

#### 検索機能

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| query検索が動作する | `?query=test` で検索実行 | 200 |
| スペース区切りのAND検索が動作する | `?query=test company` でAND検索 | 200 |

### GET /api/quotes/customers/search

#### 認証・権限チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは得意先検索を実行できない | セッションなしでアクセス拒否 | 401 |
| 権限のないユーザーは得意先検索を実行できない | 見積権限なしで拒否 | 403 |
| アクセス区分00（全て）のユーザーは実行できる | 管理者はアクセス可能 | 200 |
| sales.quotes.create権限を持つユーザーは実行できる | 見積権限ありでアクセス可能 | 200 |

#### 検索機能

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| customer_cdパラメータが動作する | `?customer_cd=001` で検索 | 200 |
| customer_nameパラメータが動作する | `?customer_name=テスト` で検索 | 200 |
| 両方のパラメータを指定できる（AND条件） | `?customer_cd=001&customer_name=テスト` | 200 |

#### レスポンス構造

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 結果にcustomer_id, customer_cd, customer_nameが含まれる | レスポンス構造の検証 | 200 |

**レスポンス例**:
```json
{
  "success": true,
  "customers": [
    {
      "customer_id": 1,
      "customer_cd": "00001",
      "customer_name": "テスト商事"
    }
  ]
}
```

## SectionCustomerApiTest.php

**ファイル**: `backend/tests/Feature/SectionCustomerApiTest.php`

部署別得意先管理APIのテスト。

### エンドポイント

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/api/customers/suggest` | 得意先サジェスト（センター指定） |
| GET | `/api/customers/section-customers` | 部署別得意先一覧 |
| POST | `/api/customers/section-customers` | 部署別得意先追加 |
| DELETE | `/api/customers/section-customers` | 部署別得意先削除 |

### GET /api/customers/suggest

#### 認証チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは得意先サジェストを取得できない | 「認証が必要です」エラー | 401 |
| 認証済みユーザーは得意先サジェストを取得できる | アクセス可能 | 200 |

#### バリデーション

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| center_idは必須 | center_id未指定でエラー | 422 |

#### 検索機能

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| query検索が動作する | `?center_id=1&query=test` | 200 |
| customer_cd検索が動作する | `?center_id=1&customer_cd=001` | 200 |
| customer_name検索が動作する | `?center_id=1&customer_name=テスト` | 200 |

### GET /api/customers/section-customers

#### 認証チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは部署別得意先一覧を取得できない | 「認証が必要です」エラー | 401 |
| 認証済みユーザーは部署別得意先一覧を取得できる | アクセス可能 | 200 |

#### バリデーション

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| center_idは必須 | center_id未指定でエラー | 422 |
| 無効なper_pageはバリデーションエラー | `per_page=999` でエラー | 422 |
| 無効なsort_orderはバリデーションエラー | `sort_order=invalid` でエラー | 422 |

#### ページネーション・ソート

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| ページネーションパラメータが正しく動作する | `page=1&per_page=25` | 200 |
| ソート順（asc）が動作する | `sort_order=asc` | 200 |
| ソート順（desc）が動作する | `sort_order=desc` | 200 |

**レスポンス構造**:
```json
{
  "success": true,
  "customers": [...],
  "total": 100,
  "page": 1,
  "per_page": 25,
  "total_pages": 4
}
```

### POST /api/customers/section-customers

#### 認証・権限チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは部署別得意先を追加できない | 「認証が必要です」エラー | 401 |
| 権限のないユーザーは部署別得意先を追加できない | 「この操作を行う権限がありません」 | 403 |
| アクセス区分00のユーザーは追加APIにアクセスできる | 200/404/409のいずれか | - |
| sales.orders.customer権限を持つユーザーはアクセスできる | 403以外 | - |

#### バリデーション

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| center_idは必須 | center_id未指定でエラー | 422 |
| customer_idは必須 | customer_id未指定でエラー | 422 |

### DELETE /api/customers/section-customers

#### 認証・権限チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは部署別得意先を削除できない | 「認証が必要です」エラー | 401 |
| 権限のないユーザーは部署別得意先を削除できない | 「この操作を行う権限がありません」 | 403 |
| アクセス区分00のユーザーは削除APIにアクセスできる | 200/404のいずれか | - |
| sales.orders.customer権限を持つユーザーはアクセスできる | 403以外 | - |

#### バリデーション

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| center_idは必須 | center_id未指定でエラー | 422 |
| customer_idは必須 | customer_id未指定でエラー | 422 |

#### エラーケース

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 存在しないセンターで削除しようとすると404 | 存在しないcenter_idで404 | 404 |
