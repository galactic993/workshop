# センターAPIテスト仕様

**ファイル**: `backend/tests/Feature/CenterApiTest.php`

## 概要

センター一覧取得APIのテスト。
アクセス権限によるセンター絞り込みの検証を含む。

## エンドポイント

| メソッド | パス | 説明 |
|---------|------|------|
| GET | `/api/centers` | アクセス可能なセンター一覧 |
| GET | `/api/centers/all` | 全センター一覧 |

---

## GET /api/centers

アクセス権限に応じたセンター一覧を取得。

### 認証チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーはセンター一覧を取得できない | セッションなしでアクセス拒否 | 401 |

**エラーレスポンス**:
```json
{
  "success": false,
  "message": "ログインしてください"
}
```

### 正常系

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 認証済みユーザーはセンター一覧を取得できる | セッションありでセンター一覧取得 | 200 |
| アクセス区分00のユーザーは全センターを取得できる | 管理者は全センターにアクセス | 200 |
| 一般ユーザーはアクセス可能なセンターのみを取得できる | 所属センターに限定 | 200 |

**レスポンス構造**:
```json
{
  "success": true,
  "centers": [
    {
      "department_id": 1,
      "department_name": "東京センター"
    },
    {
      "department_id": 2,
      "department_name": "大阪センター"
    }
  ]
}
```

---

## GET /api/centers/all

全センター一覧を取得（権限に関わらず全センター）。

### 認証チェック

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 未認証ユーザーは全センター一覧を取得できない | セッションなしでアクセス拒否 | 401 |

### 正常系

| テストケース | チェック内容 | HTTPステータス |
|-------------|-------------|---------------|
| 認証済みユーザーは全センター一覧を取得できる | セッションありで全センター取得 | 200 |
| 一般ユーザーでも全センター一覧を取得できる | 権限に関わらず全センター | 200 |
| 全センター一覧はis_center=trueのセンターのみを返す | チームは除外 | 200 |

**レスポンス構造**:
```json
{
  "success": true,
  "centers": [
    {
      "department_id": 1,
      "department_name": "東京センター"
    }
  ]
}
```

---

## アクセス権限によるセンター絞り込み

| アクセス区分 | 取得可能範囲 |
|-------------|-------------|
| 00（全て） | 全センター |
| 10（ディレクター） | 担当センター |
| 20（所長） | 所属センター |
| 30（リーダー） | 所属チーム親センター |
| 40（一般） | 所属チーム親センター |
