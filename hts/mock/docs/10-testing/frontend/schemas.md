### 日付範囲チェック

| テストケース | チェック内容 |
|-------------|-------------|
| 開始日 <= 終了日は許容する | `from='2024-01-01', to='2024-01-31'` が成功する |
| 開始日 = 終了日は許容する | 同一日付が成功する |
| 開始日 > 終了日でエラー | `from='2024-01-31', to='2024-01-01'` でエラー「開始日は終了日以前の日付を入力してください」 |
| 片方のみ入力でも許容する（開始日のみ） | 終了日が空でも成功 |
| 片方のみ入力でも許容する（終了日のみ） | 開始日が空でも成功 |

### 空文字列の許容

| テストケース | チェック内容 |
|-------------|-------------|
| 全フィールド空文字列で有効 | デフォルト値（全て空）で `success: true` |

### QUOT_STATUS定数

| テストケース | チェック内容 |
|-------------|-------------|
| QUOT_STATUS_CODE: 7つのステータスコードを持つ | 00〜60の7ステータス |
| QUOT_STATUS_CODE: 正しいコード値を持つ | REGISTERED='00', APPROVED='50', ISSUED='60' など |
| QUOT_STATUS_LABEL: 全てのコードに対応するラベルを持つ | '00'='登録済', '50'='承認済' など |
| QUOT_STATUS_OPTIONS: 8つの選択肢を持つ | '全て' + 7ステータス |
| QUOT_STATUS_OPTIONS: 最初の選択肢は「全て」 | `{value: '', label: '全て'}` |
| QUOT_STATUS_OPTIONS: 全ての選択肢がvalue/labelプロパティを持つ | 構造の検証 |

## 3. sectionCustomerSchema.test.ts

**ファイル**: `frontend/src/schemas/sectionCustomerSchema.test.ts`

部署別得意先フォームのZodバリデーションスキーマのテスト。

### center_id

| テストケース | チェック内容 |
|-------------|-------------|
| 空文字列でエラー | センター未選択でエラー「センターを選択してください」 |
| 値がある場合は許容する | センター選択済みで成功 |

### customer_query と customer_id の関係

| テストケース | チェック内容 |
|-------------|-------------|
| customer_query が空なら customer_id は不要 | クエリ空なら得意先ID不要 |
| customer_query がスペースのみなら customer_id は不要 | スペースのみは空として扱う |
| customer_query に値がある場合、customer_id が必須 | クエリ入力時は得意先選択必須、エラー「得意先を選択してください」 |
| customer_query に値があり、customer_id も設定されていれば許容する | 両方設定で成功 |

### customer_id

| テストケース | チェック内容 |
|-------------|-------------|
| 正の整数は許容する | 1以上の整数が成功する |
| nullは許容する（customer_queryが空の場合） | クエリ空なら `null` 許容 |
| 0以下でエラー | 0でエラー |
| 負の数でエラー | -1でエラー |
| 小数でエラー | 1.5でエラー |

### 有効なフォームデータ

| テストケース | チェック内容 |
|-------------|-------------|
| センター選択のみで有効 | センターのみ選択で成功 |
| センターと得意先の両方選択で有効 | 両方選択で成功 |

## 4. orderImportSchema.test.ts

**ファイル**: `frontend/src/schemas/orderImportSchema.ts`

受注情報取込フォームのZodバリデーションスキーマ。

### file（Excelファイル）

| テストケース | チェック内容 |
|-------------|-------------|
| ファイル未選択でエラー | FileListが空でエラー「取込データを選択してください」 |
| .xlsx ファイルは許容する | `.xlsx` 拡張子が成功する |
| ファイルサイズ1MB以下は許容する | 1,048,576バイト以下が成功する |
| ファイルサイズ1MB超でエラー | 1,048,577バイト以上でエラー「ファイルサイズは1MB以下にしてください」 |
| .xls や他の形式でエラー | `.xls`, `.csv` など他形式でエラー「Excel形式（.xlsx）のファイルを選択してください」 |
| MIMEタイプで検証される | `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` で成功 |
