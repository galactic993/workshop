# 見積ダイアログ仕様書

## 概要

見積ページで使用されるモーダル・ダイアログの仕様書。

## 詳細情報モーダル

### 概要

見積書No.をクリックすると表示される詳細情報モーダル。見積の詳細情報を表示し、ステータスに応じた操作ボタンを提供する。

### 基本情報表示

各項目はラベルを【】で囲み、ラベルと値を横並びで表示する。2項目を1行に並べて表示する。値がnullの場合は「-」を表示する。

#### 表示項目一覧

| 行 | 左項目 | 右項目 |
|----|--------|--------|
| 1 | 【部署】部署コード + 部署名 | 【担当者】担当者の社員名 |
| 2 | 【見積No】見積書番号 | 【得意先】得意先コード + 得意先名 |
| 3 | 【見積件名】見積件名 | 【品名】品名 |
| 4 | 【主管センター】主管センター名 | 【ステータス】ステータスラベル |
| 5 | 【見積金額】見積金額（¥カンマ区切り） | 【見積日】見積日（YYYY/MM/DD形式） |
| 6 | 【提出方法】提出方法ラベル | 【見積結果】見積結果ラベル |
| 7 | 【参考資料】参考資料パス | 【見積書格納先】常に「-」を表示 |

#### 見積概要・失注理由（別行表示）

| 項目 | 表示条件 | スタイル |
|------|---------|---------|
| 【見積概要】 | 常時表示 | ラベルの下に枠線付きで表示 |
| 【失注理由】 | 見積結果が失注（20）の場合のみ表示 | 見積概要と同様のスタイル |

## 見積テーブル（制作見積済以降）

制作見積済（prod_quot_status=20）以降で見積内訳テーブルを表示する。

| 列名 | 内容 |
|------|------|
| 作業部門コード | 3桁のコード（例: 001, 002, 003） |
| 作業内容 | 印刷、編集、開発など |
| 基準価格 | 基準価格（右寄せ、¥カンマ区切り） |
| 見積金額 | 見積金額（入力欄または表示のみ） |

### ラベル表示

| 条件 | ラベル |
|------|--------|
| prod_quot_status=20（制作見積済）かつ quot_status=00（作成中） | 【制作見積】 |
| quot_status=10（承認待ち）, 20（承認済）, 30（発行済） | 【見積】 |

### 見積金額表示（詳細モーダル）

詳細モーダルでは全ステータスで見積金額は閲覧専用表示。

| 条件 | 表示形式 |
|------|----------|
| prod_quot_status=20（制作見積済）以降 | 値表示のみ（編集不可） |

### 見積金額入力（登録・更新画面）

登録・更新画面ではステータスに応じて編集可否が変わる。

| 条件 | 表示形式 |
|------|----------|
| prod_quot_status=20（制作見積済）, quot_status=10（承認待ち） | 入力欄（編集可能） |
| quot_status=20（承認済）, 30（発行済） | 値表示のみ（編集不可） |

**入力欄の動作:**
- 初期値: ¥0
- フォーカス時: 円記号を外して数値のみ表示
- フォーカス外れ時: 円記号 + 3桁区切りで表示
- 入力値変更時: 合計金額をリアルタイム更新

## フッターボタン（詳細モーダル）

詳細モーダルは情報表示に特化しており、フッターには「閉じる」ボタンのみを表示する。

| ボタン | 説明 |
|--------|------|
| 閉じる | モーダルを閉じる |

**注意**: 詳細モーダルでは編集・操作機能を提供しない。編集が必要な場合は一覧の「選択」ボタンから登録・更新画面へ遷移する。

### 制作見積依頼実行確認ダイアログ

制作見積依頼モーダルで「制作見積依頼」ボタンを押した場合に表示される確認ダイアログ。

| 項目 | 値 |
|------|-----|
| タイトル | 制作見積依頼 |
| 本文 | 制作見積依頼を行います。よろしいですか？ |
| キャンセルボタン | キャンセル |
| 確認ボタン | 制作見積依頼 |

**ボタン動作:**

| ボタン | 動作 |
|--------|------|
| キャンセル | ダイアログを閉じる（モーダルは表示したまま） |
| 制作見積依頼 | 制作見積依頼APIを実行し、成功時は成功ダイアログを表示 |

**実行処理:**
- `POST /api/quotes/{quotId}/request-production` を呼び出し
- 見積テーブル（quots）の `prod_quot_status` を `10`（制作見積依頼済）に更新
- 制作見積テーブル（prod_quots）にレコードを作成
- メール送信処理を実行（送信先の所長宛て）

### 制作見積依頼成功ダイアログ

## 流用作成モーダル

### 概要

過去の発行済見積から得意先・主管センター情報を流用して新規見積を作成するためのモーダル。新規登録モードまたはステータスが「作成中」の場合にヘッダーの「流用作成」ボタンから開く。

### 表示条件

- 新規登録モード（`/sales/quotes/new`）
- 詳細モードでステータスが「作成中」（`quot_status=00`）の場合

| ボタン | 説明 |
|--------|------|
| キャンセル | モーダルを閉じる |
| 選択 | 選択した見積の情報を流用してフォームにセット（見積未選択時は非活性） |

### 流用処理

「選択」ボタン押下時に以下の項目を登録フォームにセット:

| 項目 | 動作 |
|------|------|
| 得意先 | 流用元の得意先をセット |
| 得意先名（手入力） | 流用元が諸口の場合、見積得意先名をセット |
| 主管センター | 流用元のセンターをセット |
| 流用元見積ID | 流用元の見積IDを保持（非表示、登録時にbase_quot_idとして送信） |
| 作業部門別見積 | 流用元の作業部門別見積を読み取り専用テーブルで表示 |

### 作業部門別見積の流用

流用元の見積に作業部門別見積（quot_operations）が存在する場合、以下のテーブルで表示する:

```
【作業部門別見積】
┌────────────────┬──────────┬───────────┬─────────────────┐
│ 作業部門コード │ 作業内容 │ 基準価格  │ 見積金額        │
├────────────────┼──────────┼───────────┼─────────────────┤
│ 001            │ 印刷     │ ¥50,000   │ ¥60,000         │
│ 002            │ 編集     │ ¥30,000   │ ¥35,000         │
├────────────────┴──────────┼───────────┼─────────────────┤
│                  合計金額  │ ¥80,000   │ ¥95,000         │
└─────────────────────────────────────────────────────────┘
```

| 列名 | 内容 |
|------|------|
| 作業部門コード | 作業部門の3桁コード |
| 作業内容 | 作業部門名 |
| 基準価格 | 流用元の基準価格（cost）、右寄せ・¥カンマ区切り |
| 見積金額 | 流用元の見積金額（quot_amount）、右寄せ・¥カンマ区切り |

**注意事項:**
- テーブルは読み取り専用で表示
- 見積登録時に `base_quot_id` と `operations` 配列としてAPIに送信
- 新規見積に紐づく `quot_operations` レコードとして新規作成される
- 再度流用作成モーダルから別の見積を選択した場合、前回の作業部門別見積はクリアされ新しいデータに置き換わる

### 詳細モーダル表示

検索結果の見積書Noをクリックすると、見積一覧の詳細モーダルと同じ内容が表示される。流用作成モーダルの上にz-index: 60で表示。

### 状態リセット

モーダルを閉じた際に以下の状態がリセットされる:
- 検索フォームの入力値
- 検索結果
- 選択状態
- 詳細モーダルの状態

## z-indexの制御

モーダルの重なりを適切に制御するため、z-indexを調整している。

| 要素 | z-index |
|------|---------|
| 詳細モーダル / 流用作成モーダル / 制作見積依頼モーダル | z-50 |
| 確認ダイアログ / 詳細モーダル（流用作成モーダルから表示） / 制作見積依頼実行確認ダイアログ / 制作見積依頼成功ダイアログ | z-[60] |
