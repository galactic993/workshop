# フロントエンド デザインシステム

## 概要

管理会計システムのフロントエンドにおけるデザインシステムを定義します。
一貫性のあるUI/UXを提供するための基準となるデザイントークン、スタイリング規則を記載しています。

---

## レイアウト

### 画面幅

**固定幅レイアウト**: 1366px

```css
max-width: 1366px;
```

- 画面幅が1366pxを超える場合、左右に余白を表示
- コンテンツは中央配置（`margin: 0 auto`）
- レスポンシブ対応は将来的に検討

### 余白と背景

| 要素 | 背景色 | 説明 |
|------|--------|------|
| **余白部分** | `#d1d5db` (bg-gray-300) | 画面外側の余白エリア |
| **メインコンテンツ** | `#ffffff` | 1366px幅のコンテンツエリア |

## タイポグラフィ

### フォントファミリー

**メインフォント**: Meiryo UI

```ts
// tailwind.config.ts
fontFamily: {
  sans: ['"Meiryo UI"', 'Meiryo', 'sans-serif'],
}
```

- Windowsシステムフォントを使用
- フォールバック: Meiryo → sans-serif

### テキストサイズ

| 用途 | Tailwind Class | サイズ | 使用例 |
|------|---------------|--------|--------|
| 大見出し | `text-3xl` | 30px | - |
| 見出し | `text-2xl` | 24px | ヘッダータイトル |
| 中見出し | `text-xl` | 20px | セクションタイトル |
| 本文（大） | `text-lg` | 18px | 重要な本文 |
| 本文 | `text-base` | 16px | 標準テキスト |
| 本文（小） | `text-sm` | 14px | ブレッドクラム、補足 |
| キャプション | `text-xs` | 12px | 注釈、ラベル |

### テキストカラー

```tsx
// メインテキスト
className="text-gray-900"

// 補足テキスト
className="text-gray-600"

// セパレーター
className="text-gray-400"
```

## コンポーネント共通仕様

### ボーダー

**標準ボーダー**:
```tsx
className="border-b border-gray-200"
```

- 太さ: 1px
- 色: `#e5e7eb` (gray-200)

### シャドウ

**軽い影（コンテンツエリア）**:
```tsx
className="shadow-sm"
```

### トランジション

**ホバー効果**:
```tsx
className="transition-colors"
```

- デュレーション: 150ms (Tailwind デフォルト)
- イージング: ease-in-out

## アクセシビリティ

### コントラスト比

- テキスト色と背景色のコントラスト比は WCAG AA 基準（4.5:1以上）を満たすこと
- 現在の組み合わせ:
  - gray-900 on white: ✅ 16.1:1
  - gray-600 on white: ✅ 7.2:1

### フォーカス表示

インタラクティブ要素には明確なフォーカススタイルを適用:
```tsx
className="focus:outline-none focus:ring-2 focus:ring-blue-500"
```
