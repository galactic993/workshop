# 加工品内容コードマスタ (proces_cds)

## 概要

**テーブル物理名**: `proces_cds`

**テーブル論理名**: 加工品内容コードマスタ

**用途**:
- 実際の作業項目を管理
- 加工品内容コードの先頭3桁で作業部門マスタにグループ化
- 制作見積詳細で使用される工程の基準単価を定義

## テーブル定義

| No. | 論理名 | 物理名 | データ型 | 桁数 | Not Null | PK | UK | FK | Default | 備考 |
|-----|--------|--------|----------|------|----------|----|----|----|---------| ----- |
| 1 | 加工品内容コードマスタid | proces_cd_id | BIGSERIAL | - | ● | ● | ● | - | - | 自動採番 |
| 2 | 加工品内容コード | proces_cd | CHAR(8) | 8 | ● | - | - | - | - | 半角数字8桁 |

## 加工品内容コード体系

加工品内容コードは8桁の数字で構成され、先頭3桁で作業部門にグループ化される。

| 先頭3桁 | 作業部門 | コード範囲 |
|---------|---------|-----------|
| 100 | 印刷 | 10000010 〜 10099999 |
| 200 | 編集 | 20000010 〜 20099999 |
| 300 | 開発 | 30000010 〜 30099999 |
