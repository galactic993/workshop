# s1.5-reference-spec スキル

## 概要
画面設計書の参照仕様シートに記載されたテーブル名・カラム名が、テーブル定義書（共・売・編）に実在するか検証する。

## 使い方
python ~/.claude/skills/s1.5-reference-spec/scripts/check_reference_spec.py <設計書ディレクトリ> [--output result.xlsx]

## チェック内容
1. テーブル定義書3ファイルから全テーブル論理名・カラム論理名の辞書を構築
2. 参照仕様シートから◆セクション内の抽出テーブル名・抽出項目名を抽出
3. テーブル名・カラム名をそれぞれ辞書と突合
4. 存在しないテーブル/カラムへの参照を検出

## 終了コード
- 0: 全PASS
- 1: WARN/FAILあり
