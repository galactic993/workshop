# s1.5-reference-spec スキル

## 概要
画面設計書の参照仕様シートに記載されたテーブル名・カラム名が、テーブル定義書（共・売・編）に実在するか検証する。

## 使い方

`shared-agent-skills` では `scripts/run-session1-claude-p.sh` に画面設計書・テーブル定義書を渡すと、本スキルを含む s1.5 プロンプトが並列実行されます。

単体で使う場合は本ディレクトリの `SKILL.md` を `~/.claude/skills/` に置き、Claude / Claude Code で Excel を読み取れるように指示してください。

## チェック内容
1. テーブル定義書3ファイルから全テーブル論理名・カラム論理名の辞書を構築
2. 参照仕様シートから◆セクション内の抽出テーブル名・抽出項目名を抽出
3. テーブル名・カラム名をそれぞれ辞書と突合
4. 存在しないテーブル/カラムへの参照を検出

## レビュー上の判定
- **PASS**: 参照先がテーブル定義書に存在
- **WARN/FAIL**: 未定義参照または要確認
