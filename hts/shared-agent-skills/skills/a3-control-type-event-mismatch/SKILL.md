# A3: 種別とイベントの矛盾チェック

**重要度:** error
**対象シート:** 項目記述書

## プロンプト

```
項目記述書シートで、種別とイベント番号の整合性をチェックしてください。

【項目記述書】
- ヘッダーは4行目（B列=No., G列=項目名, R列=種別, Y列=EVENT No.）

以下の種別はイベント番号が必須です：
- ボタン
- リンク
- プルダウン
- ラジオボタン

チェック内容：
1. R列（種別）が上記のいずれかで、かつY列（EVENT No.）が空欄の行を検出
2. これらはクリック/選択時にイベントが発生するため、イベント番号が必要

結果を以下の形式で出力してください：
| 行番号 | 項目名 | 種別 | EVENT No. | チェック結果 |
|--------|--------|------|-----------|--------------|
例：| 20 | 検索ボタン | ボタン | （空欄） | エラー: イベント番号未設定 |
```

## 説明

このプロンプトは、項目の種別に応じたイベント番号の設定確認です。

- **対象種別**: ボタン、リンク、プルダウン、ラジオボタン
- **チェック内容**: これらの種別の項目には、クリックまたは選択時に発火するイベント番号が必須

## 使用上の注意

- 表示専用テキストやテキストインプット等の種別では、イベント番号が必須ではありません
- エラーが検出された場合は、必ずイベント番号を設定してください
