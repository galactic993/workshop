# s1.5-screen-capture スキル

## 概要

画面設計書の画面概要シートに埋め込まれたキャプチャ画像内の項目名・ボタンラベルと、項目記述書の項目名リストを突合する。

## 使い方

`shared-agent-skills` では `scripts/run-session1-claude-p.sh` に画面設計書・テーブル定義書を渡すと、本スキルを含む s1.5 プロンプトが並列実行されます（画像の読み取りは `claude` が行います）。

単体で使う場合は本ディレクトリの `SKILL.md` を `~/.claude/skills/` に置き、Claude / Claude Code で Excel と埋め込み画像を読み取れるように指示してください。

## チェック内容

1. 画面概要シートから埋め込み画像を確認する
2. 画像から項目名・ボタンラベルを読み取る
3. 項目記述書の項目名リストと突合する
4. 画像側のみ／項目記述書側のみの項目や表記ゆれを検出する

## 注意

- `claude` CLI が必要な場合があります。利用できない場合はスキップしてレポートにその旨を記載してください。
- 画像が埋め込まれていない場合は、画面概要シートのテキスト情報で代替チェックを実施してください。

## レビュー上の判定

- **PASS**: 突合に問題なし
- **WARN/FAIL**: 差分あり、画像なし、または要確認
