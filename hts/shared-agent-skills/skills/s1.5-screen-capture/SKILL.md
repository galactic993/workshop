# s1.5-screen-capture スキル

## 概要

画面設計書の画面概要シートに埋め込まれたキャプチャ画像内の項目名・ボタンラベルと、項目記述書の項目名リストを突合する。

## 使い方

`shared-agent-skills` では `scripts/run-all-check.sh` に画面設計書・テーブル定義書を渡すと、本スキルを含む s1.5 プロンプトが並列実行されます（画像の読み取りは `claude` が行います）。

単体で使う場合は本ディレクトリの `SKILL.md` を `~/.claude/skills/` に置き、Claude / Claude Code で Excel と埋め込み画像を読み取れるように指示してください。

## チェック内容

まず **画面概要シートに埋め込み画像があるか**を確認する（ツールによっては `_images` 相当で件数取得）。

### A. 埋め込み画像がある場合

1. 画像から項目名・ボタンラベルを読み取る
2. 項目記述書の項目名リストと突合する
3. 画像側のみ／項目記述書側のみの項目や表記ゆれを検出する

### B. 埋め込み画像がない場合（0件）

1. **WARN** を前提に、画面概要シート上の **テキスト・箇条書き・ラベル説明**だけで、項目記述書の項目名と可能な範囲で突合する
2. テキストでも突合不能な場合は、**画像なしのため検証不能**と明記する

## 注意

- `claude` CLI が必要な場合があります。利用できない場合はスキップしてレポートにその旨を記載してください。
- 画像なしで **PASS** にしない（原則 **WARN**、テキストで十分整合が取れた場合のみ **PASS** を検討）。

## レビュー上の判定

- **PASS**: 画像またはテキスト代替で突合に問題なし
- **WARN/FAIL**: 差分あり、画像なしで代替不十分、または要確認
