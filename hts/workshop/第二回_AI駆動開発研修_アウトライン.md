# 第二回 AI駆動開発研修 アウトライン

## 1. 研修の目的

- 第一回で学んだ「設計書レビュー・Markdown化」の流れを、実装フェーズに接続できる状態にする
- AIコーディング、AIテスト、AIレビューを一連の開発サイクルとして実践し、受講者が現場に持ち帰れる運用手順を確立する
- エージェント活用時の品質担保ポイント（指示品質・検証・レビュー観点）を共通化する

## 2. 到達目標

- 受講者が Claude Code / Codex のいずれかで、要件から実装・テスト・レビューまでを自走できる
- 生成コードに対して、最小限必要な検証（テスト、差分レビュー、再現確認）を実施できる
- チームで再利用できるレビュー観点チェックリストを作成できる

## 3. タイムライン付きアジェンダ（120分）

| 時間 | セッション | 形式 | 狙い |
|---|---|---|---|
| 13:30 - 13:40 | オープニング・第一回の振り返り | 講義 | 前回内容と今回テーマの接続を明確化 |
| 13:40 - 14:00 | AIコーディング実践デモ | デモ | 指示から実装までの進め方を可視化 |
| 14:00 - 14:30 | 演習1: AIコーディング | ハンズオン | 小さな機能追加をエージェントで実装 |
| 14:30 - 14:40 | 休憩 | - | 集中力リセット |
| 14:40 - 15:00 | AIテスト設計・実行 | 講義+デモ | 回帰防止を意識したテスト観点を整理 |
| 15:00 - 15:20 | 演習2: AIテスト・AIレビュー | ハンズオン | テスト作成、変更差分へのレビューと修正ループを実践 |
| 15:20 - 15:30 | まとめ・質疑応答 | 講義 | 実務適用のアクションを明確化 |

## 4. 演習内容

### 演習1: AIコーディング（30分）

- お題: 既存機能に対する小規模改善（例: 入力バリデーション追加、表示文言の条件分岐）
- 実施内容:
  - 要件を箇条書きで定義し、エージェントに実装を指示
  - 変更ファイルの差分確認と意図確認
  - 必要に応じて再指示して修正
- 完了条件: 仕様を満たす実装と、変更理由を説明できる状態

### 演習2: AIテスト・AIレビュー（20分）

- お題: 演習1で追加した機能に対するテスト作成とレビュー
- 実施内容:
  - テストケース（正常系/異常系/境界値）を定義
  - エージェントにテストコード作成と実行を依頼
  - レビュー観点（可読性・保守性・仕様適合性）で指摘を出し、再修正
- 完了条件: テストが通り、主要レビュー指摘が解消されている状態

## 5. 期待成果物

- 演習対象機能の実装差分（変更ファイル一覧と要約付き）
- 最低1本以上の追加テスト（正常系/異常系を含む）
- 受講者チームのレビュー結果メモ（指摘内容・修正内容・判断理由）
- 次回までに現場適用するためのアクションリスト（3項目以上）

## 6. 準備物

### 受講者側

- Claude Code または Codex が動作する開発環境
- サンプルプロジェクト（第一回で使用した設計書Markdownと同一リポジトリ）
- 画面共有可能な端末と Slack 接続

### 講師側

- デモ用リポジトリ（成功例/失敗例の両方を準備）
- テスト失敗を再現できるサンプルケース
- タイムキープ用の進行シート

## 7. 進行メモ

- 前半デモは「一発で成功」よりも「修正ループの見せ方」を重視する
- 演習中は受講者が詰まるポイントを 10 分ごとに回収し、全体共有で解決する
- レビューでは「なぜその指摘が必要か」を必ず言語化させる
- 時間超過時は演習2をミニケース化し、レビュー観点の説明を優先する

## 8. 評価観点

| 観点 | 評価基準 | 確認方法 |
|---|---|---|
| 指示品質 | 要件・制約・完了条件が明確に指示できている | プロンプト記録の確認 |
| 実装品質 | 仕様を満たし、既存機能への影響が抑えられている | 差分レビュー |
| テスト品質 | 正常系/異常系/境界値の観点が押さえられている | テストコードと実行結果 |
| レビュー品質 | 指摘が具体的で、修正方針に反映されている | レビューコメントと再差分 |
| 振り返り品質 | 現場適用アクションが具体化されている | アクションリスト確認 |

## 9. 研修後フォロー

- 1週間以内に Slack で実務適用結果を共有
- 講師は共有内容に対して優先度付きでフィードバック
- 代表的な成功/失敗パターンを第三回導入資料に反映
