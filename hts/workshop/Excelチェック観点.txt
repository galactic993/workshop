画面設計書チェック（A系：内部整合性）

  ┌─────┬───────────────────────────────────────────────────────────────────────────────────────────────┬───────────────┐
  │ ID  │                                         チェック内容                                          │    重要度     │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A1  │ イベント番号の連番性（EVENT0001, 0002...の抜け番検出）                                        │ warning       │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A2  │ イベント番号のクロスシート対応（項目記述書のEVENT No.がイベント記述書に存在するか）           │ error         │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A3  │ 種別とイベントの矛盾（ボタン/リンク/プルダウン/ラジオボタンにイベント番号が未設定）           │ error         │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A4  │ 種別とバリデーションの矛盾（表示専用テキストに制御内容がある/テキストインプットに制御が無い） │ error/warning │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A5  │ メッセージの存在確認（項目記述書のメッセージがメッセージ一覧に存在するか）                    │ error  (（不要）)     │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A6  │ 項目番号の連番性                                                                              │ warning       │
  ├─────┼───────────────────────────────────────────────────────────────────────────────────────────────┼───────────────┤
  │ A7  │ メッセージ番号の連番性                                                                        │ warning       │
  └─────┴───────────────────────────────────────────────────────────────────────────────────────────────┴───────────────┘
