# Claude in Excel 用プロンプト集

このドキュメントは、Excel の Claude in Excel サイドバー（Ctrl+Option+C）で使用するプロンプト集です。
各チェック項目について、そのままコピペして使用できるプロンプトを提供します。

---

## クイック参照表

| ID | チェック内容 | 重要度 | 対象ファイル |
|----|--------------|--------|--------------|
| A1 | イベント番号の連番性 | warning | 画面設計書のみ |
| A2 | イベント番号のクロスシート対応 | error | 画面設計書のみ |
| A3 | 種別とイベントの矛盾 | error | 画面設計書のみ |
| A4 | 種別とバリデーションの矛盾 | error/warning | 画面設計書のみ |
| A5 | メッセージの存在確認 | error（不要） | 画面設計書のみ |
| A6 | 項目番号の連番性 | warning | 画面設計書のみ |
| A7 | メッセージ番号の連番性 | warning | 画面設計書のみ |
| B1 | DB登録値のフィールド名存在 | error | 画面設計書 + テーブル定義書 |
| B2 | 参照仕様のテーブル.カラム存在 | warning | 画面設計書 + テーブル定義書 |
| C1 | FK参照先テーブル/カラム存在 | error | テーブル定義書のみ |
| C2 | PK定義チェック | warning | テーブル定義書のみ |
| C3 | NOT NULLかつデフォルト値未設定 | info | テーブル定義書のみ |

## 使用上の注意点

### Claude in Excel の起動方法
- Mac: `Ctrl + Option + C`
- Windows: `Ctrl + Alt + C`

### 効果的な使用方法
1. **単一チェックごとに実行**: 複数のチェックを一度に行うと出力が長くなるため、1つずつ実行することを推奨
2. **ヘッダー行の確認**: 各シートのヘッダー行番号が異なる場合、プロンプト内の行番号を調整してください
3. **空行の除外**: データ抽出時に空行が含まれる場合があります。 Claude は自動的に除外しますが、気になる場合はプロンプトで指定してください

### エラー対処
- 「シートが見つかりません」エラー: シート名が正確かつ確認してください
- 「データが抽出できません」エラー: ヘッダー行番号を確認してください

---

## D系 使い分けガイド

| プロンプト | 用途 | 前提条件 |
|-----------|------|----------|
| D1 | 画面設計書の単体Markdown化 | 画面設計書のみ開く |
| D2 | テーブル定義書の単体Markdown化 | テーブル定義書のみ開く |
| D3 | 画面とテーブルを統合した設計書 | 両方を開く |

**推奨ワークフロー:**
1. まずD1とD2を個別に実行して、各ファイルのMarkdownを確認
2. 問題なければD3で統合版を作成
3. 出力されたMarkdownをコピーし、`.md` ファイルとして保存
4. Claude Codeに読み込ませて開発作業に使用
