## AI・LLMとは

### LLM（大規模言語モデル）の概要

LLM（Large Language Model）は、大量のテキストデータから学習した人工知能モデルです。人間の言語を理解し、文章の生成、翻訳、要約、コード生成などを行います。

### 主要なLLMの種類

| LLM | 開発元 | 特徴 |
|-----|--------|------|
| Claude | Anthropic | 安全性重視。長文理解に強い。コーディング能力が高い |
| ChatGPT / OpenAI | OpenAI | 最も普及。マルチモーダル対応。Thinking / Proではブラウザ操作型のリサーチも強い |
| Gemini | Google | 画像生成はできるが、現時点では無理に使う優先度は高くない |
| Grok | xAI | X連携と最新話題のリサーチに強い。AI関連の情報収集で試す価値が高い |

### 主要なコーディングエージェント

| ツール | 開発元 | 形態 | 特徴 |
|--------|--------|------|------|
| **Claude Code** | Anthropic | CLI / IDE / デスクトップ | 最も多機能。MCP連携、サブエージェント |
| **Codex** | OpenAI | Web / API | クラウドサンドボックス実行。プラグイン追加でブラウザ操作も可能 |
| **Cursor** | Cursor Inc. | 専用エディタ | AI特化エディタ。複数モデル切替 |
| **GitHub Copilot** | GitHub / MS | IDE拡張 | 最も普及。GitHub連携 |

### リサーチ用途の補足

- OpenAIのThinkingモード / Proモード、およびGrokは、実際にブラウザを操作しながら情報収集できる
- 通常のスクレイピングでは取得しにくい情報でも、ブラウザ経由の操作で到達できることがある
- とくにAI関連やX上の最新動向を追うなら、Grokを使ってみるのがおすすめ
- OpenAIのエージェントモードでもブラウザ操作は可能だが、少し不安定な場面がある
- Claude CodeやCodexも、別途プラグインを入れることでブラウザ操作が可能になるため、ブラウザを触らせたいときはこちらを推奨する

## AIの課題と対応のベストプラクティス

### ハルシネーション（幻覚）
AIが事実に基づかない情報を生成する現象。もっともらしいが間違った回答をすることがある。

**対策:**
- 重要な情報は必ず人間が検証する
- セルフフィードバックループ（AIの出力をAIに検証させる）
- 複数回のやり取りで精度を上げる

### セキュリティ

| プラン | データの学習利用 | 備考 |
|--------|----------------|------|
| 無料 | される可能性あり | 設定でOFF可の場合も |
| 有料個人 | 設定次第 | OFFにすれば使われない |
| API / Team / Enterprise | されない | 商用規約で明確に禁止 |

**ベストプラクティス:**
- APIキーは他人に教えない（クレジットカードと同等）
- 本番の機密データはAIに入力しない
- 有料/APIプランを使用する

## プロンプトエンジニアリング入門

### プロンプトとは
AIに対する「指示文」のこと。プロンプトの品質がAIの出力品質を決定する。

### 良いプロンプトの基本要素

1. **役割の指定** — AIに何者として振る舞うか指示する
2. **具体的なタスク** — 何をしてほしいか明確に記述
3. **出力形式の指定** — どのような形式で回答してほしいか
4. **制約条件** — 守るべきルールや条件
5. **コンテキスト** — 背景情報や前提条件

### 例：悪いプロンプト vs 良いプロンプト

**❌ 悪い例:**
```
アプリを作って
```

**✅ 良い例:**
```
営業担当者が使う顧客管理ツールを作成してください。

要件:
- 顧客名、電話番号、メールアドレスを登録できる
- 一覧画面で検索・フィルタリングが可能
- スマートフォンでも操作できるレスポンシブデザイン

技術スタック:
- フロントエンド: Next.js + Tailwind CSS
- バックエンド: Supabase
- デプロイ: Vercel
```

## 研修で使用するツール

### Claude CLI
Anthropic社のCLIベースのコーディングエージェント。ターミナルから直接AIに作業を任せることができる。

### Vercel
フロントエンドのデプロイプラットフォーム。GitHubと連携して自動デプロイが可能。
